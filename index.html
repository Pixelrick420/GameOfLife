The game of life is not a conventional game with players and win conditions, but a simulation consisting of a grid of cells with simple rules goverining the evolution of patterns formed by these cells.

In the game, a cell can either be alive or dead. The state of being alive is denoted by 1 (green) and the dead cells are zeros (red).

Each state of the board is like a moment in time. The state of the board in the next moment depends on 4 rules.
Any live cell with fewer than two live neighbours dies.
Any live cell with two or three live neighbours lives on to the next generation.
Any live cell with more than three live neighbours dies.
Any dead cell with exactly three live neighbours becomes a live cell.

Because we are going to mars.(real)

This game is an example of cellular automation. Given any possible cell state, it is impossible to predict what the board will look like after say, 50 steps without actually running the simulation. Put more rigourously, given 2 board states A and B, there exisits no algorithm that can predict whether state B will appear at any point when state A is simulated.

The game of life is Turing complete. This means that the game can perform any computation that a Turing machine can. So.. yes, given enough cells, you can build a computer inside the game. Going back to the game of life, how do you actually make self replicating systems? Consider machine A that can build copies of machine B. A must be more complex than B, right? John von Neumann, in his 2D system for cellular automata, found a particular pattern would self replicate. This design is called the tessellation model. It consisted of a configuration of 200,000 cells and 29 board states. So, in this case machines A and B are the same.

But this was complex and messy. Conway wanted to find a much simpler system that could exhibit the same self replicating behaviour. Well, he did it.

The game of life achieves universal cellular automatation to the point where the game itself can be built within the game.

John Conway's Game of Life is not a conventional game with players and win conditions. It has however, like any popular game, captured the intrest of millions of people around the world.
